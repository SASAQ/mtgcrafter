# MTG Crafter

MTG プレイヤー向けのデッキビルダーアプリです。デッキの作成・管理から、一人回しでのテストプレイまで、MTG プレイヤーに必要な機能をシンプルな UI で提供します。

## 主な機能

- **カード検索**: Scryfall データベースに対応した詳細検索
- **デッキ作成・管理**: 直感的なデッキエディタ、サイドボード対応
- **デッキビジュアル**: 3 種類のレイアウトでデッキを可視化、SNS シェア対応
- **一人回し**: 実戦感覚のテストプレイ機能、ライブラリ操作・マナプール管理
- **MTG Crafter Pro**: 月額 400 円で広告非表示・無制限デッキ作成
